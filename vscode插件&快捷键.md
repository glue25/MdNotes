yapf在默认状态的快捷键：

```
ctrl+shift+i
```

切换到编辑区
```
ctrl+1 (打开第二个编辑区就是ctrl+2)
```

切换到终端
```
ctrl+alt+1
```

开新终端
```
ctrl+shift+`
```

在编辑器页标签切换
```
ctrl+pageup
ctrl+pagedown
```
保持打开状态

```
ctrl+k enter
```

固定标签页

```
ctrl+k shift+enter
```

关闭标签页

```
ctrl+w
```







单个折叠

```
ctrl+shift+[
```
单个打开折叠
```
ctrl+shift+]
```
递归折叠
```
ctrl+k ctrl+[
```
递归展开
```
ctrl+k ctrl+]
```
折叠级别
```
ctrl+k ctrl+数字1-7
```
全部折叠
```
ctrl+k ctrl+0
```
全部展开
```
ctrl+k ctrl+j
```



折叠侧边

```
ctrl+b
```

折叠终端

```
ctrl+`
```

资源管理器

```
ctrl+shift+e
```
